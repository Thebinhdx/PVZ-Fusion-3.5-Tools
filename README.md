# PVZ Fusion 3.5 Toolkit
## Description
A collection of Cheat Engine scripts designed to enhance the gameplay experience for Plants vs. Zombies Fusion 3.5. These scripts manipulate game memory to provide utility features like unlimited sun and instant recharge.

## Features
- No Cooldown: Instant recharge for all plants.

- Boost Sun: Manage sun resources effectively.

## Requirements
- Cheat Engine 7.x (Latest version recommended).

- Plants vs. Zombies Fusion 3.5 (This table is specifically optimized for this version).

## How to use
- 1.Download and install Cheat Engine.

- 2.Open the game.

- 3.Open the PVZ_Fusion_3.5.ct file using Cheat Engine.

- 4.Select the game process (the monitor icon in Cheat Engine).

- 5.Tick the checkboxes for the cheats you want to activate.
